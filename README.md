# 🧠 Quiz-deutsch – Python Quiz Spiel

Ein einfaches Konsolen-Quiz in Python, bei dem der Spieler Fragen beantworten und Punkte sammeln kann.  
Das Spiel bietet zwei Schwierigkeitsstufen: **Normal** und **Schwer**.

---

## ⚙️ Funktionen

- Eingabe des Spielernamens
- Zwei Schwierigkeitsstufen:
  - Normal
  - Schwer
- Punktesystem
- Verschiedene Wissensfragen (Geografie, Politik, Allgemeinwissen)
- Eingaben werden automatisch bereinigt (.strip().lower())

---

## 🕹️ So funktioniert das Spiel

1. Programm starten
2. Namen eingeben
3. Schwierigkeitsgrad wählen
4. Fragen beantworten
5. Am Ende werden die Punkte angezeigt
